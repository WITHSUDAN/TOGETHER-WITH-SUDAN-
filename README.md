# TOGETHER-WITH-SUDAN-
مبادرة "معاً من أجل السودان" الشعار: "بسواعدنا نبني وطننا" أولاً: فكرة المبادرة. هي مبادرة وطنية تطوعية تهدف إلى توحيد الجهود السودانية الرسمية والشعبية داخل السودان وخارجه، لدعم الاستقرار، الإغاثة، والتنمية، وتعزيز روح التضامن الوطني في مواجهة العدوان، والانقسام، والدمار الناتج عن الحرب.
